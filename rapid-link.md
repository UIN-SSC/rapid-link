## 📝 Panduan Mengelola Tautan (Untuk Admin)

Website ini berjalan menggunakan HTML murni agar super ringan dan cepat. Jika di kemudian hari Anda ingin **menambahkan link baru**, memperbarui file `.md` ini saja **tidak** akan otomatis mengubah website. Anda harus menambahkan kodenya ke dalam file `index.html`.

Berikut adalah panduan dan template yang bisa Anda _copy-paste_ ke dalam `index.html` (letakan di dalam area `<section class="links-section">`):

### Template HTML untuk Link Baru:
```html
<a href="MASUKKAN_URL_DISINI" class="link-card" target="_blank" rel="noopener noreferrer">
    <div class="link-icon"><i class="fa-solid fa-link"></i></div>
    <div class="link-content">
        <h3>MASUKKAN JUDUL TAUTAN DISINI</h3>
    </div>
    <div class="link-arrow"><i class="fa-solid fa-chevron-right"></i></div>
</a>
```

**Tips Ikon:** Anda bisa mengganti teks `fa-link` di atas dengan ikon lain dari [FontAwesome Free](https://fontawesome.com/search?o=r&m=free). 
- PDF: `fa-file-pdf`
- WhatsApp: `fa-whatsapp` (gunakan `fa-brands fa-whatsapp`)
- Instagram: `fa-instagram` (gunakan `fa-brands fa-instagram`)
- Website/Portal: `fa-globe` atau `fa-newspaper`
